# 秋元康アリーナ

「AIが秋元康風の歌詞を書く」サービス。どのAI（あるいはどのプロンプト）が、最も「秋元康」を憑依させられたかを競うアリーナです。

## 機能

- 複数のAIモデルを並行実行して秋元康風の歌詞を生成
- ユーザーによる投票・順位付け
- ベンチマークの自動生成

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。
